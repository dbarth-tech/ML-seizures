# ML-seizures
Matlab code for seiure detection using machine learning
requires MatLab Machine Learning Toolbox

two scripts are available
1) "train" = read example data for SRS, SWD and NOISE   look inside the script to see data format
2) "classify" = read single channels of EEG data, senent into q sec blocks, and classify seizure type

3) a few additional subroutines caled by these scripts are also uploaded here
